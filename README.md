# my-weather-app

##### Android Application developed as a pre qualification exercise for the Mobile Software Development Intern position at HomeSpotter.
---
##### How to build the project:

1. Download the source code

2. Open the project with Android Studio

3. Using the SDK Manager (Tools-> Android) to download the latest version of Android Support Library

4. Hit the "Sync Project with Gradle Files" button

5. Make the Project (Build -> Make Project)

6. Run it

---
##### Some project highlights:

- Support for multiple cities
- Settings page to define units, date and hour format
- App Widget

---
##### Some features used:

- RecyclerViews
- CardViews
- Navigation drawer
- ViewPager
- SwipeRefreshLayout
- Settings activity
- App Widget
- IntentService
